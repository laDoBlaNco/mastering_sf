# The Trailmaker | Salesforce Developer Series - 3 | Types of Programming Languages

---

There are two types of programming languages
- Low-level Language
    - **Maching Language**: Machine dependent, difficult to modify or program. For example, every CPU has its own machine language. The code written in machine language is the instruction set that the processor uses
    - **Assembly Language**: Each computer's microprocessor that is responsible for arithmetic, logical and control activities needs instructions for accomplishing such tasks and these instructions are written in assembly language. The use of assembly language is in device drivers, low-level embedded systems, and real-time systems.  
- High-level Language
    - Independent of the hardware
    - Their code is very simple and developers can read, write and debug it as its similar to everyday English statements.
    - High-level languages can  be further divided into 3 categories:
        - **Procedural Languages**: Code in the procedural category is a sequential step by step procedure, that gives information to the device on *what* and *how* to do something. Languages such as Fortran, Cobol, Basic, C, and Pascal are exaamples of procedural languages
        - **Non-procedural Languages**: Code in non-procedural language (declarative) specifies *what* to do, but not *how* to do it. Examples are SQL, Prolog, LISP, etc.
        - **Object Oriented Languages**: These languags use objects, where code is built to reflect real-life objects and its more about manipulating data such as properties and state. Examples are C++, Java, Ruby, and Python.
