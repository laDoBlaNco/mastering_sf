# The Trailmaker | Salesforce Developer Series - 2 | What is a Computer

---

A computer can be broken down into 4 things:
- input device
    - Keyboard
    - Mouse
    - gamepad/joystick
    - microphone
    - digital camera
    - scanner, etc
- output device
    - screen/monitor
    - printer
    - speaker, etc.
- memory and storage
    - Random Access Memory (RAM)
        - used only while the device is powered up
        - small and faster (thinking power)
    - Secondary/permanent storage space (HDD/SSD)
        - retain data after device is powered down. Long-term memory
        - Slower and larger
    - Read Only Memory (ROM)
        - stores instructions used to boot/start the computer or device (BIOS)
        - smart devices that don't get reprogrammed (IOT, printers, washing machines, etc) have some class of ROM in them with instructions on what to do and how to work
- processing
    - CPU (MicroProcessor)
    - The brain of your computer or device
