# The Trailmaker | Salesforce Developer Series - 4 | Basic Programming Concepts

---

- **Algorithm**: In the most general sense, an algorithm is a series of instructions telling a computer how to transform a set of facts about the world into useful information. The facts are data, and the useful informatijon is knowledge for people, instrutions for machines or input for yet another algorithm.
<br>

- **Source Code**: Source code is the actual text that is used to construct the program using the language of choice
<br>

- **Compiler**: A compiler is a software program that converts the source code into bytecode or binary (machine language), for a computer to understand. It can also be executed using an interpreter (in the case of bytecode). Compilers understand the **syntax** of whatever language you are using.
<br>

- **Date Type**: Data used in the applications can be of different types. For example, whole numbers (Integer), floating point  numbers (Decimal), characters (String), etc
<br>

- **Variable**: Variable is a space holder for the value stored in the memory and this value can be used in the application
<br>

- **Statement Blocks**: {...} anything between two curly braces (*in certain languages like Apex and Java*) is called a stagement block or block of statements
<br>

- **Conditionals**: If a condition is true, do something, if it is false, do something else or do nothing.
<br>

- **Collection**: A variable to store multiple values of the same data type. There are a few diffent collection types:
    - *Array/List*: a variable to store multiple values of the given data type
    - *Set*: A variable to store multiple **unique** values of the given data type.
    - *Map*: A variable that stores multiple values as key=>value pairs of the given data types for the key and a different type for the value. Since Map keys must be unique, some would say that we have a *Set* of keys connected to a *List* of values.
<br>

- **String**: A collection of characters or an array of characters by definition. But, a lot of modern languages help us to play around with strings just like any other data type.
<br>

- **Loop**: Loops are used  to execute a series of code multiple times either to a certain count or until a condition changes or until a colleciton is empty as well.
<br>

- **Function**: Functions are used to reuse a piece of code whenever required at any place in our source code. They do a particular task, can take input as parameters/arguments, can return desired output or just complete some action.
    - This is key for Apex and Salesforce as requirements change constantly so all code needs to be scalable and easy to change.
<br>

- **Class**: Class is a template or blueprint that contains state and behavior, which corresponds to a field/property and a method/function. In Object Oriented Programming languages like C++, Jave, Apex, etc, everything revolves around Classes and Objects
<br>
