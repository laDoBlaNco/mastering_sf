---

### What is Apex?

- Apex is a the first multi-tenant, on-demand programming language, which some would call an extension of Java and others would not.
- Its for developers interested in building the next generation of business applications. 
- Revolutionizes the way that developers create on-demand (data) applications
- Apex sits on the same cloud instance as our data. Apex joins what used to be your language vs database connection calls all into one thing. 
- Apex is strongly-typed (that is, I must declare the data type of a variable when first referring to it).
- Object oriented language that allows developers to execute flow and transaction control statements on Salesforce servers in conjunction with calls to the API
- Using syntax that is similar to Java and behaves like database stored procedures, Apex enables developers to add business logic to most system events, including button clicks, relatd record updates, and Visualforce pages.
- Apex code can be initiated by a web service request and from triggers on objects
- All statements must end in a ';'
